## Run Locally

Clone the project

```bash
  git clone https://github.com/gedharizka/automation-api.git
```

Go to the project directory

```bash
  cd automation-api
```

Install dependencies

```bash
  npm install
```

run testing

```bash
  npm run testing
```
run report

```bash
  npm run report
```


## Tech Stack

**Testing framework:** Mocha

**Assertion library:** Chai

**Reporter:** Mochawesome

**HTTP Request:** supertest
